# Helper function to confirm that the argument B to several functions is as expected

Helper function to confirm that the argument B to several functions is
as expected

## Usage

``` r
checkB(B)
```

## Arguments

- B:

  Integer or numeric; the number of subsamples. Note: For
  sampling.type=="MB" the total number of subsamples will be `B`; for
  sampling_type="SS" the number of subsamples will be `2*B`.
