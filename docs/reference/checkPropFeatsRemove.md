# Helper function to confirm that the argument prop_feats_remove to several functions is as expected

Helper function to confirm that the argument prop_feats_remove to
several functions is as expected

## Usage

``` r
checkPropFeatsRemove(prop_feats_remove, p)
```

## Arguments

- prop_feats_remove:

  Numeric; proportion of features that are dropped on each subsample.
  Must be between 0 and 1.

- p:

  The number of features; must be greater than 2 if prop_feats_remove is
  greater than 0.
