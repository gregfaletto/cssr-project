# Helper function to confirm that the argument cutoff to several functions is as expected

Helper function to confirm that the argument cutoff to several functions
is as expected

## Usage

``` r
checkCutoff(cutoff)
```

## Arguments

- cutoff:

  Numeric; only those clusters with selection proportions equal to at
  least cutoff will be selected by cluster stability selection. Must be
  between 0 and 1.
