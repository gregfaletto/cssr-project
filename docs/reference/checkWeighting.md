# Helper function to confirm that the argument weighting to several functions is as expected

Helper function to confirm that the argument weighting to several
functions is as expected

## Usage

``` r
checkWeighting(weighting)
```

## Arguments

- weighting:

  Character; determines how to calculate the weights to combine features
  from the selected clusters into weighted averages, called cluster
  representatives. Must be one of "sparse", "weighted_avg", or
  "simple_avg'.
